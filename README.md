# Node js TypeORM Demo

> DB 參數請在 ormconfig.json 設定
> nodemon 參數請在 nodemon.json 設定

<br>

如何使用本專案<br>

1. cd your_path/your_project
2. npm install
3. 本機測試: npm run dev
4. Debug: npm run dev:debug
5. 機器運行: npm run start
